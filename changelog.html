Recent changes to CIVS

Most users use the existing CIVS web server. However, the software is available for installation elsewhere, subject to certain restrictions described in the included README file. To download the latest released version of the CIVS software, click here:

See the file INSTALL for installation instructions.

Recent updates:

  • 2.7:
    • Added ability to release election results to a limited set of users.
    • Better client-side sanity checking for election creation.
    • Allow previewing of election description and choice list.
    • Added another completion algorithm (runoff) and factored out completion algorithm code better to allow future completion algorithms to be added more easily.
    • Allow voters' identity to be revealed (an election option).
    • Mail to voters is now HTML+ASCII (should be made optional).

The current snapshot of the source tree is also available. Note that the source tree often contains code that has not yet been pushed to the server and that may not be ready for general release.
